# Stylesheets

CSS files defining the Digital Bytes design system and website styling.

## Files

| File | Purpose |
|------|---------|
| `theme.css` | Primary theme stylesheet |
| `theme.min.css` | Minified production version |
| `custom-theme.css` | Custom theme overrides |
| `custom-accessibility.css` | Accessibility enhancements and WCAG compliance |
| `fonts.css` | Font loading and typography definitions |

## Design System

- **Brand Color:** #5746e3 (Primary Purple)
- **Font:** Inter (sans-serif)
- **Responsive:** Mobile-first, tablet, and desktop
- **Accessibility:** WCAG 2.1 AA compliant
